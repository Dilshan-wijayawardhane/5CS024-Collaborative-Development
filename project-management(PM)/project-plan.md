Project Plan Report


Project Title: Synergy_hub
Team Name: SYNERGY_SIX 
Duration: 10 Weeks


Introduction

This project aims to deliver a Campus Companion System that enhances student engagement and convenience. Beyond the MVP rewards system, the updated scope includes unique features such as a live campus map, smart notifications, and sustainability tracking. These additions transform the system into a holistic platform that supports academic, social, and lifestyle needs of students.
Objectives
Create a functional rewards-based check-in system. Integrate facility and event feeds into a student dashboard. Offer transport and club hub functionality for students. Implement special features like give below
Live Campus Map with GPS navigation and facility markers.
Smart Notifications for events, transport, and reminders. 
Sustainability Tracker to reward eco‑friendly actions.
Gamified Challenges to boost engagement.
Crowd Density Display: Use GPS checkins to estimate how busy a facility is (e.g., 	“Gym is 70% full”).
Deliver examiner‑ready documentation and a demonstration video.
